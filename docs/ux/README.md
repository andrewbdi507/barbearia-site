## 🎯 Princípios de Design

1. **Mobile First** — 90% dos clientes usam celular. Projetar para polegar primeiro.
2. **Zero Fricção** — Cada etapa removida é uma conversão ganha.
3. **Invisível** — O cliente não sente que usa um "sistema".
4. **Óbvio** — Sem manual. Sem tutorial. Sem dúvida.
5. **Rápido** — Agendamento em < 2 minutos. Tela carrega em < 2 segundos.
6. **Acessível** — WCAG 2.1 AA. Funciona para todos.
7. **Consistente** — Mesmo padrão em todas as telas, todos os dispositivos.
